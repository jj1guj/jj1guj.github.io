#### Slackにひたすら気象情報を投下するボット
   サークルのSlackで毎日決まった時刻につくば市の3時間予報投下できたら便利そうだなぁと思って色々調べて作ってみました  
    Incoming Webhook, OpenWeathermapを使用し、Python3で実装しています  
    天気予報を投下するつもりが現在は1時間毎に今の天気を垂れ流すだけになっているのでそろそろなんとかしたいです\(\)
